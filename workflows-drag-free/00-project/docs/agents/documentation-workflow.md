# Documentation Workflow (workflows-drag-free)

- Long-lived user/project docs: `00-project/docs/`
- Agent-facing conventions: `00-project/docs/agents/`
- Operational change history: `00-project/changelog/`
- Issues worth preserving: `00-project/troubleshooting/`
- Active plans: `00-project/plans/`; completed: `00-project/plans-completed/`

Root `AGENTS.md` stays slim and links here. Prefer updating the changelog for every material change.
