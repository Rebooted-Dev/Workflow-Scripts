# Project Structure (workflows-drag-free)

## Meta root

`workflows-drag-free/00-project/` is the **flattened project meta** for the drag-free workflow tree. There is no nested `project/` wrapper; changelog, plans, docs, and troubleshooting live directly under `00-project/`.
