## Boundary: `00-docs/` vs `00-meta/`

| Directory | Content | Examples |
|-----------|---------|----------|
| **`00-meta/`** | Templates, rubrics, and **design documents about the workflow system** | Severity rubric, glossary, sync template, agent flexibility guidelines |
| **`00-docs/`** | **Generated reports and archived references** | Code review reports, release records, retired runbooks, conflict logs |

**Rule of thumb:** If it's a current reusable template or design reference → `00-meta/`. If it's a one-time generated report or historical reference → `00-docs/`.
