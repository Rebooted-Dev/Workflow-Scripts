# Plans (Workflow-Scripts)

This is the **Workflow-Scripts** plans index for ongoing work on the workflow repo itself (e.g. workflow revisions, docs, remediation). Active plans live here; completed/archived plans go to [00-plans-completed/](../00-plans-completed/) with a date prefix and index entry.

**Project repos** that run the setup (e.g. Info-Visualizer) use a different structure: they have a **`project/`** directory (KIV, research, build, changelog, troubleshooting) and **`plans/`** at project root with README (map to project dir) and TODO.md. Active plan documents live in `project/build/`; completed plans go to `project/changelog/plans/` with a row in `project/changelog/index.md` (Type=plan). See the project's `plans/README.md` for the full map.
