# Plans

Active implementation plans and planning documents live here.

When a plan is completed or archived, move it to [plans-completed/](../plans-completed/) and rename the file with a `yyyy-mm-dd-` prefix (e.g. `2026-02-28-my-plan.plan.md`). Then add a row at the top of [plans-completed/index.md](../plans-completed/index.md) with the date, title/description, and file link.
